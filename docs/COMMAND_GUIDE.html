PlanX CartoLab — Command Guide v1.4

All 12 Processing algorithms in the PlanX CartoLab provider. Each entry includes the algorithm ID (for PyQGIS) and its purpose.

Classification

geometric_interval_classification

Classify a numeric field using one of three advanced algorithms: Adaptive GIC (optimal for skewed continuous data), Head/Tail Breaks (heavy-tailed / power-law distributions), or Fisher-Jenks natural breaks. Adds gic_class and gic_label fields.

Thematic Mapping

bivariate_choropleth

Create a bivariate choropleth map by classifying two numeric fields into an N×N colour matrix. Adds bivar_x_class, bivar_y_class and bivar_class fields. Output is auto-styled with the computed bivariate colour matrix.

ridge_map

Generate a ridge-line (Joy Division style) vector layer from a single-band raster. Raster values deform scanlines vertically, producing overlapping wave profiles. Best with transparency and a dark canvas background.

value_by_alpha

Encode uncertainty / reliability as opacity (alpha channel). Primary variable drives colour; secondary reliability variable controls opacity. Adds vba_alpha (0–255) and vba_alpha_pct (0–100) fields.

dot_density

Scatter dots inside each polygon, one dot per N units of a count field. Placement is seeded and hole-aware. Each dot inherits source polygon attributes for multi-group dot maps.

proportional_symbols

Place a point at each feature centroid whose symbol size is proportional to a numeric field. Flannery compensation corrects perceptual area under-estimation. Adds psym_size (mm) for data-defined marker size.

Cartogram

compute_cartogram

Distort polygon areas to be proportional to a numeric field using the diffusion method (Gastner & Newman). Iteratively displaces polygon boundaries until each region's area represents its field value.

2.5D Styling

building_25d_style

Apply a polished native QGIS 2.5D renderer to a loaded polygon layer. Controls roof/wall colour, shadow, viewing angle, height clamp, stepped extrusion, and per-floor colour bands. Supports height-in-metres or floor-count modes.

Aggregation

hexbin_aggregate

Aggregate a point layer into a pointy-top hexagonal grid. Only occupied hexagons are emitted. Statistic: count, sum, or mean of a weight field. Output carries hex_count, hex_sum and hex_mean.

Labeling

label_points

Compute each polygon's pole of inaccessibility (polylabel) — the interior point furthest from any edge. Always falls inside the polygon, even for C-shaped or doughnut features. Adds lbl_dist (inscribed-circle radius).

Map Reference

graticule_grid

Generate a line layer of meridians and parallels across an extent, on nice round coordinate intervals. Each line carries its orientation, constant coordinate and a formatted label. Leave interval at 0 to auto-pick.

Data Preparation

normalize_field

Turn a raw field into a map-ready value: Rate (numerator / denominator × scale), z-score, robust z (median-MAD), min-max (0–1), percentile rank (0–100), or log₁₀. Writes norm_value and norm_method fields.
